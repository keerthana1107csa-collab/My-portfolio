# Personal Portfolio Website

## 📌 Project Overview
This project is a simple and responsive personal portfolio website created using HTML5 and CSS3.  
The website demonstrates semantic HTML structure, accessibility features, SEO-friendly design, and responsive layout techniques.

---

## 🚀 Features
- Semantic HTML5 structure
- Responsive design using CSS
- Accessible navigation and forms
- SEO meta tags
- Multi-page website
- Contact form
- Clean and modern UI

## ▶️ How to Run the Project

1. Download or clone the project
2. Open the folder in Visual Studio Code
3. Install the Live Server extension
4. Right click `index.html`
5. Click **Open with Live Server**
